#### 1. 基本概念
+ **简介：** 所有元素都会在插入时自动被排序
+ **本质：** set/multiset属于**关联式容器**，底层结构是用**二叉树**实现的
+ **区别：**
  + set 不允许容器中有重复的元素
  + multiset 允许容器中有重复的元素
